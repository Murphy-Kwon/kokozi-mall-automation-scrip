### 코코지 몰 장바구니 개선 과제

코코지몰 장바구니 기능 개선 과제를 수행하며 작성한 테스트 케이스 중 일부 항목에 대해 **UI 자동화 테스트 적용 가능성을 검토**하였습니다.

이 중 **CART-11 (상품 장바구니 담기 Alert 노출 확인)** 항목을 자동화 테스트 예시로 선정하여 정리하였습니다.

---

| 항목 | 내용 |

---

### 테스트 목표

상품 상세 페이지에서 **[장바구니에 담기] 버튼을 클릭했을 때** 장바구니 담기 완료 Alert이 정상적으로 노출되는지 확인합니다.

---

### 테스트 시나리오

1. 코코지몰 접속  
2. 상품 클릭 → 상품 상세 페이지 진입  
3. **[장바구니에 담기] 버튼 클릭**  
4. 장바구니 담기 완료 Alert 확인  

---

### 기대 결과

장바구니 담기 완료 Alert이 노출되며 다음 요소가 확인된다.

- **"장바구니에 담았습니다!" 안내 문구**
- **[쇼핑 계속하기] 버튼**
- **[장바구니로 이동] 버튼**

---

### 자동화 접근 배경

해당 테스트는 UI 상에서 반복적으로 검증이 필요한 기능이므로 브라우저 기반 자동화 테스트 도구를 활용한 검증이 가능하다고 판단하였습니다.

본 저장소에서는 **Playwright 기반 UI 자동화 테스트 구조를 참고하여 작성한 예시 스크립트**를 정리하였습니다.

---

### 자동화 스크립트 설명

본 자동화 스크립트는 테스트 흐름을 설명하기 위한 예시 코드입니다.  
실제 테스트 환경에서는 locator 정의, 테스트 데이터 구성, 그리고 다양한 브라우저 환경에 대한 검증이 추가로 필요할 수 있습니다.  
또한 UI 변경에 따라 테스트 스크립트 수정이 필요할 수 있습니다.
