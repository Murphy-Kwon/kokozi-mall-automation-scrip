Test Case : CART-11
Test Name : 장바구니 담기 Alert 확인

테스트 목적
상품 상세 페이지에서 [장바구니에 담기] 버튼 클릭 시
장바구니 담기 완료 Alert이 정상적으로 노출되는지 확인한다.

테스트 과정
1. 코코지몰 접속
2. 상품 클릭
3. 상품 상세 페이지 진입
4. [장바구니에 담기] 버튼 클릭
5. 장바구니 담기 Alert 노출 확인
6. Alert 내 버튼 확인

테스트 결과

Step 5 결과

PASS (성공)
"장바구니에 담았습니다!" Alert 메시지가 정상적으로 노출된다.

FAIL (실패)
Alert 메시지가 노출되지 않거나, 메시지 내용이 요구사항과 다르게 표시된다.


Step 6 결과

PASS (성공)
Alert 내 다음 두 개의 버튼이 정상적으로 표시된다.

- 쇼핑 계속하기
- 장바구니로 이동

FAIL (실패)
Alert 버튼이 표시되지 않거나 클릭이 불가능하다.

Pseudo Automation Flow (Playwright style)

1. Open kokozi mall website
   → navigate to https://kokozi.com

   → click product item

   FAIL

6. Verify alert buttons
   → Continue Shopping

   FAIL
